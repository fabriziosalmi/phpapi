ncov2

Description

Retrieve worldwide daily sum of confirmed cases, deaths and fatality rate of the COVID19 disease (data source).

How to use

https://phpapi.org/ncov2/php.api?date=mm-dd-YYYY&country=xxxxxx

Time range: from 01-22-2020 to yesterday (required)

Example

Input

https://phpapi.org/ncov2/php.api?date=03-03-2020&country=Italy

Output

{"date":"03-03-2020","country":"Italy","confirmed":"2502","deaths":"79","ratio":3.16}
