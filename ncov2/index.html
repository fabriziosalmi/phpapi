ncov2

Description

Retrieve worldwide daily sum of confirmed cases, deaths and fatality rate of the COVID19 disease (data source).

How to use

https://phpapi.org/ncov2/php.api?date=mm-dd-YYYY

Time range: from 01-22-2020 to yesterday (required)

Example

Input

https://phpapi.org/ncov2/php.api?date=03-05-2020

Output

{"date": "03-05-2020","confirmed": 97886,"deaths": 3349,"fatality": 3.42}
